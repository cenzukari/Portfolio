# Portfolio

В портфолио отобраны 4 реализованных мной проекта. 

| #    | Наименование проекта                | Описание                                                     | Стек                                                         |
| ---- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 1.   | [Предсказание температуры стали](https://github.com/cenzukari/Portfolio/tree/main/Steel%20Temperature) | Построение модели для предсказания температуры стали для уменьшения затрат электроэнергии | python, pandas, numpy, seaborn, sklearn, matplotlib, catboost       |
| 2.   | [Восстановление золота из золотосодержащей руды](https://github.com/cenzukari/Portfolio/tree/main/Gold%20recovery) | Подготовка прототип модели машинного обучения, которая должна предсказать коэффициент восстановления золота из золотосодержащей руды| python, pandas, numpy, matplotlib, seaborn, sklearn |
| 3.   | [Прогноз количества заказов для сервиса такси](https://github.com/cenzukari/Portfolio/tree/main/Taxi%20Order%20Forecast) | Построение модели для прогнозирования количества заказов такси на следующий час для привлечения большего числа водителей в период пиковой нагрузки            | python, pandas, numpy, matplotlib, sklearn, statsmodels, catboost, lightgbm |
| 4.   | [Определение стоимости автомобилей с пробегом](https://github.com/cenzukari/Portfolio/tree/main/Car%20Cost) | Построение модели, которая будет определять рыночную стоимость автомобиля в приложении для клиентов сервиса по продаже автомобилей с пробегом           | python, pandas, numpy, seaborn, matplotlib, sklearn, catboost, lightgbm   |
